# Notes Taking App

A clean and simple notes-taking web app built with HTML, CSS, and JavaScript.

## Features

Create notes
Edit and save note title/content
Delete notes
Search notes by title or content
Auto-persist notes in browser `localStorage`
Responsive layout for desktop/mobile

## Project Structure

`index.html` – App layout and semantic structure
`styles.css` – App styling and responsive UI
`app.js` – Notes logic, rendering, and localStorage persistence

## Run locally

Because this is a static app, you can open `index.html` directly in your browser.

Or run a small local server:

```bash
python3 -m http.server 8000
```

Then open: "https://localhost:8000";
